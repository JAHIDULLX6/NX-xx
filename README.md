## ✨ About

**Mirai Bot** is a modern and modular bot framework designed for Facebook Messenger automation. It supports admin tools, fun commands, media features, and easy customization for developers.

---

## 🚀 Features

- ⚡ Fast and lightweight  
- 🧩 Modular command system  
- 👑 Admin / moderation commands  
- 🎵 Media and download commands  
- 🖼️ Image tools & AI support  
- 🛠️ Easy configuration  
- 📦 Supports VPS / Render / Railway / Replit  
- 🔁 Auto-reload supported
